## Introduction

This project contains the HTML and CSS code for Sourav Miranam's personal portfolio website. It showcases his profile, experience, projects, and contact information.

## Sections

1. **Profile Section**: Displays Sourav's profile picture, name, and a brief introduction.

2. **About Section**: Provides detailed information about Sourav's background, education, and experience.

3. **Experience Section**: Highlights Sourav's experience in frontend and backend development, listing the technologies he's familiar with.

4. **Projects Section**: Showcases Sourav's recent projects with images, links to their GitHub repositories, and live demo links.

5. **Contact Section**: Provides ways to get in touch with Sourav, including email and LinkedIn links.

6. **Footer**: Contains navigation links and copyright information.

## Usage

You can use this code as a template for your own personal portfolio. Feel free to customize it according to your preferences.

To view the website locally, open the `index.html` file in a web browser.
